
# Transaction Data Analysis Project

## Project Overview
This project focuses on analyzing a transactional dataset to understand sales performance, customer behavior, product demand, and regional trends. The dataset contains transaction details such as transaction amount, customer ID, service category, product type, state, city, and payment details.

## Objective
The goal of this project was to perform data analysis using Python and Pandas to answer various business-related questions and extract meaningful insights from the data.

## Approach

1. Loaded the dataset using Pandas.
2. Converted transaction date into datetime format.
3. Extracted month and quarter from transaction dates.
4. Performed groupby operations for aggregation.
5. Calculated total, average, and maximum transaction values.
6. Analyzed customer purchasing patterns.
7. Evaluated product and service performance.
8. Conducted state-wise and city-wise sales analysis.
9. Compared spending patterns based on payment methods.

## Key Analysis Performed

### Sales Analysis
- Total sales across all transactions
- Monthly and quarterly sales trends
- Highest single transaction amount
- Month and quarter with highest revenue

### Customer Analysis
- Average transaction per customer
- Total spending per customer
- Repeat buyer percentage
- Multi-category purchase behavior

### Product & Service Analysis
- Service category generating highest revenue
- Most popular services by transaction count
- Product type performance
- Underperforming categories

### Regional Analysis
- State-wise total sales
- City with highest transactions
- Service popularity across states
- Comparison between California and Texas spending

### Payment Mode Analysis
- Credit vs debit transaction comparison
- Average spending based on payment type
- Total revenue from credit transactions

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Conclusion
The analysis provided insights into customer behavior, high-performing products and services, seasonal trends, and regional sales patterns. These insights can help businesses make data-driven decisions regarding marketing, inventory management, and customer targeting strategies.
